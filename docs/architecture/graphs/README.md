# Architecture Graphs

This folder contains generated dependency graphs and reports for Dialogue Forge.

## Generated Outputs
- `dependency-cruiser.mmd`: Mermaid graph of module dependencies.
- `dependency-cruiser.d2`: D2 graph output for tooling that supports D2 diagrams.
- `madge.json`: Madge JSON dependency map.
- `../dependency-cruiser.json`: Raw dependency-cruiser JSON output (used by analysis scripts).
- `../latest-analysis.md` and `../latest-analysis.json`: Summary reports generated by `scripts/architecture-analyzer.js`.

## Regenerating the Graphs
Run these from the repo root:

```bash
# Dependency-cruiser outputs (requires TypeScript in NODE_PATH)
NODE_PATH=./node_modules npx dependency-cruiser --no-config --output-type json --output-to docs/architecture/dependency-cruiser.json src
NODE_PATH=./node_modules npx dependency-cruiser --no-config --output-type mermaid --output-to docs/architecture/graphs/dependency-cruiser.mmd src
NODE_PATH=./node_modules npx dependency-cruiser --no-config --output-type d2 --output-to docs/architecture/graphs/dependency-cruiser.d2 src

# Madge JSON map (TypeScript-aware)
npx madge --extensions ts,tsx --ts-config tsconfig.json --json src > docs/architecture/graphs/madge.json

# Summaries
node scripts/architecture-analyzer.js docs/architecture/dependency-cruiser.json
```
